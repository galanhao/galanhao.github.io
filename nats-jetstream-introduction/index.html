NATS JetStream：消息队列的现代选择

NATS JetStream 是 NATS 的持久化消息系统，提供 at-least-once 和 exactly-once 语义。

基本概念

  • Stream：消息存储单元，可配置保留策略
  • Consumer：订阅 stream 并消费消息的实体
  • Subject：消息路由的主题

Go 客户端示例

import "github.com/nats-io/nats.go/jetstream"

nc, _ := nats.Connect(nats.DefaultURL)
js, _ := jetstream.New(nc)

// 创建 stream
js.CreateStream(ctx, jetstream.StreamConfig{
    Name:     "EVENTS",
    Subjects: []string{"events.>"},
})

// 发布消息
js.Publish(ctx, "events.user.created", []byte(`{"user_id":"123"}`))

// 消费消息
cons, _ := js.CreateConsumer(ctx, "EVENTS", jetstream.ConsumerConfig{
    Durable: "processor",
    AckPolicy: jetstream.AckExplicitPolicy,
})

msgs, _ := cons.Messages()
for msg := range msgs {
    // 处理消息
    msg.Ack()
}
